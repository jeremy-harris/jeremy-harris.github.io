WVU Football Analytics

I have built a pipeline for WVU football data (although it can be adopted to any team). The pipeline uses the API from www.collegefootballdata.com and pulls in all plays from a specified game. I started with a single game and plan to expand this further to include more analysis, more games and eventually a Shiny app that will allow users to select their desired games.

Active Directory Threat Detection

This project will utilized unsupervised learning to detect anomalies in Windows Active Directory (AD) data. Once detected, the data will utilize a signature and be sent through an ML pipeline for detection of true findings while minimizing false positives. This project will be conducted in Python.

More to come on this...

COVID-19 Chill Out!

I took the Johns Hopkins data for the COVID pandemic in 2020. My goal is to take the data and visualize it in a way that compares different regions according to confirmed cases and population density. It is pretty clear that one size does NOT fit all and in my opinion, we need to consider more surgical approaches to reducing the spread and less of the brute-force 'atomic' option.
